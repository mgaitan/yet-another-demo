## Quick Start

Run directly without installing via `uvx`

```bash
uvx yet-another-demo --help
```

```{richterm} env PYTHONPATH=../src uv run -m yet_another_demo --help
:hide-command: true
```

To install the tool permanently:

```bash
uv tool install yet-another-demo
```
